# Summarize Skill

Distilling complex information into clear, actionable summaries.

## Summary Types

### Executive Summary
- 2-3 paragraphs max
- Lead with the conclusion
- Include key numbers/data
- End with recommendation

### Bullet Summary
- 5-7 key points
- One idea per bullet
- Most important first
- Action-oriented language

### Technical Summary
- Problem statement
- Solution approach
- Key trade-offs
- Implementation notes

## Guidelines
- Preserve the core meaning
- Remove redundancy
- Use simpler words
- Maintain original tone
- Keep proportional coverage

## Template
```markdown
## Summary: [Title]

**TL;DR:** [One sentence]

**Key Points:**
- Point 1
- Point 2
- Point 3

**Details:** [If needed]

**Source:** [Original document/URL]
```
