# Search Skill

Effective strategies for finding information across web and code.

## Web Search Strategies
- Use specific keywords, not natural language
- Add site: filter for trusted sources
- Use quotes for exact phrases
- Add year for recent information
- Combine terms with AND/OR

## Code Search Strategies
- Search for function/class names
- Use regex for patterns
- Filter by file type
- Search error messages verbatim
- Look in tests for usage examples

## Source Evaluation
| Factor | Questions to Ask |
|--------|------------------|
| Authority | Who wrote this? What are their credentials? |
| Currency | When was this published? Is it still relevant? |
| Accuracy | Is this verifiable? Are sources cited? |
| Purpose | Why was this written? Any bias? |

## Search Refinement
1. Start broad, then narrow
2. Use different terminology
3. Check related searches
4. Follow citation trails
5. Look for primary sources
