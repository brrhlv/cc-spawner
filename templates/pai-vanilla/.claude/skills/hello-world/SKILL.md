# Hello World Skill

A minimal example skill showing how PAI skills are structured.

## Usage

Type `/hello` or `/hello-world` to invoke this skill.

## Behavior

When invoked, greet the user and explain the PAI skill system.

## Response

Say hello and briefly explain:
1. This is a sample skill
2. Skills are defined with SKILL.md files
3. They can have triggers (slash commands)
4. They extend Claude's capabilities

Keep it friendly and concise.
